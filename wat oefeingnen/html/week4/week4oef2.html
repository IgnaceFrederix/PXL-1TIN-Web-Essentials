Competentiebeschrijving

Je leert de nodige vaardigheden om zelf een statische website te maken. Hiervoor werken we met de gangbare webstandaarden, namelijk Hypertext Markup Language (HTML) en Cascading Style Sheets (CSS). In het laatste deel krijg je een introductie tot meer geavanceerde standaarden en scriptingtalen om zo je eerste dynamische pagina's te bekomen. Het is de bedoeling dat je naast het hanteren van deze vaardigheden ook een gefundeerde keuze kan maken aan de hand van het doel van de website of onderdelen van de website. Je leert ook je code te documenteren, te onderhouden en te hergebruiken. Je verwerft vaardigheden om multimediatoepassingen te realiseren in verschillende contexten. Om je de nodige inzichten te geven in de manier waarop dit alles werkt, zal je rechtstreeks met de code werken door middel van een tekst editor. Je leert ook hoe je een website kan installeren en valideren.
